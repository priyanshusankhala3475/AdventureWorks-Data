# AdventureWorks Sales Dashboard | Power BI

An interactive Power BI dashboard built using the AdventureWorks dataset to analyze sales performance, customer behavior, and business KPIs. The dashboard provides valuable insights through dynamic visualizations, DAX measures, and Power Query transformations.

## Features

* Sales Performance Analysis
* Revenue & Profit Tracking
* Product Category Analysis
* Customer & Regional Insights
* Time-based Sales Trends
* Interactive Filters & Slicers
* KPI Cards for Business Monitoring

## Tools Used

* Microsoft Power BI
* Power Query
* DAX (Data Analysis Expressions)
* AdventureWorks Dataset

## Files Included

* `pga11.pbix` – Power BI dashboard
* `AdventureWorks-Raw-Data.zip` – Source dataset
* Dashboard screenshots

## Key Skills Demonstrated

* Data Cleaning
* Data Modeling
* Data Visualization
* DAX Calculations
* Business Intelligence
* Dashboard Design

This project demonstrates the use of Power BI to transform raw business data into meaningful insights that support data-driven decision-making.
